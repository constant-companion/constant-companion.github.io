添加分类

发表于 2021-04-18 分类于 博客

​ 在写文章的过程中，我发现上传的所有文章都展示在一个同一个列表，不便于对文章进行分类整理，查阅资料给hexo添加了分类和标签功能，实现过程如下。

首先在导航栏添加分类和标签列 进入主题配置文件，找到menu 取消里面的tags和categories的注释

然后回到源目录，运行git bash 执行命令hexo new page categories ，然后会在source文件夹里创建一个categories文件，打开里面有一个index.md，添加type: categories保存并关闭。

然后给文章添加categories属性 categories: 博客 属性值相同的会自动加入同一分类，一篇文章只能添加到一个分类中，如果是多个默认在第一分类中。

标签的生成过程类似。

rgba and hsla

发表于 2021-04-18 分类于 css

新了解到一种css色彩模式HSLA，在这里记录一下RGBA和HLSA，做一个比较。RGBA和HSLA两种色彩模式，二者均可以用来在设置颜色的同时也指定其透明度。

RGBA指的是“红色、绿色、蓝色、Alpha透明度（Red-Green-Blue-Alpha）”
前三个值取值从0255或0%100%
HSLA指的是“色调、饱和度、亮度、Alpha透明度”
色调取值0360，饱和度和亮度取值0%100%
其中，Alpha的取值从0~1 , 0代表完全透明，1代表完全不透明。

RGBA无法直观看出是什么颜色。并且如果想要对颜色进行调整也无法简单做到。
使用HSLA则只要将色调值设为一个特定值就可以方便地调整其亮度和饱和度。
对于色调值，其实就是一个色盘。

0和360对应纯红色
120对应绿色
240对应蓝色
要获取HSLA的黑色值，只要简单的将亮度设置为0%；白色则是亮度100%。对于这两个颜色，色调和饱和度可以设置为任何值。

而要获取HSLA格式的灰色，只需将饱和度设置为0%，此时亮度值控制灰色的明暗程度，色调的取值则无关紧要。

博客搭建

发表于 2021-04-15 更新于 2021-04-18 分类于 博客

博客搭建中遇到的一些问题

​ 我的博客搭建是利用hexo在github上搭建，主要根据https://godweiyang.com/2018/04/13/hexo-blog/ 这篇文章，在搭建的过程中遇到了一些问题，在这里记录一下，当做我的第一篇博客。其实也不算是博客，目前就打算记录一下自己学习的一些知识。

​ 1.安装hexo 后初始化我们的网站 输入hexo init 初始化文件夹不成功，提示Error: target not empty 网上查找后明白这是hexo的初始化必须在一个空目录下，包括隐藏的目录。发现我没有再我创建的文件夹目录下去安装，而是直接在根目录下进行的安装操作，所以用cd命令进入我创建的文件夹里面后就成功了。

​ 2.本地的网站配置好后，输入hexo g生成静态网页错误，不能打开http://localhost:4000/， 网上寻找后发现是因为在配置config.yml里面的deploy时缩进错误，里面需要缩进。改正后可以正确打开了。

​ 3.生成秘钥SSH key，这个博主的教程里并不是很清晰，可以参考https://www.cnblogs.com/ayseeing/p/3572582.html 这篇文章。

​ 4.在刚才我想发布这篇博客时又遇到了两个问题，第一个问题是里面的一些单词，hexo，github等都会自动加红色波浪线，强迫症表示看着很难受。这是我使用的Typora的问题，也在这里记录一下。左上角文件里面选择偏好设置，然后编辑器里面的拼写检查，改成不需要检查就好。第二个问题就是发布不成功，报错can not read a block mapping entry； a multiline key may not be an implicit key at line 4, column 1。查阅后发现这是因为创建的md文件头部声明必须加空格，基础的一个知识点，长知识了。添加空格后果然发布成功。
